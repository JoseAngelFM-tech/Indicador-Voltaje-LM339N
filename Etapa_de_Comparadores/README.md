# Modelado y Análisis de la Etapa de Comparadores en Python

Se desarrolló un código en Python para el modelado de la etapa de comparación analógica del sistema indicador de voltaje.  
El programa automatiza el cálculo de resistencias, niveles de voltaje e impedancia equivalente de cada comparador LM339N configurado como comparador de ventana.  
Además, permite generar gráficas para visualizar el comportamiento eléctrico del sistema y validar experimentalmente la estabilidad de las subventanas de detección.
