# Implementación Física de la Etapa de Procesamiento

Se implementó físicamente la etapa de procesamiento lógico utilizando transistores BJT, diodos y redes lógicas RTL/DTL en protoboard.
Durante esta etapa se validó experimentalmente la conversión de las señales analógicas provenientes de los comparadores de ventana hacia estados digitales, permitiendo el procesamiento lógico necesario para la identificación de diferentes niveles de voltaje dentro del sistema indicador.
